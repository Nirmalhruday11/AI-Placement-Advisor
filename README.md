# 🚀 AI Placement Advisor

An AI-powered placement recommendation platform built using **ServiceNow** and **Gemini AI** that helps students discover suitable career opportunities based on their skills, identify skill gaps, and receive personalized career guidance.

## ✨ Features

* 👨‍🎓 Candidate Profile Management
* 🛠️ Skill Inventory Tracking
* 🏢 Opportunity Catalog Management
* 🤖 AI-Powered Skill Matching
* 📊 Match Score Calculation
* 🎯 Missing Skill Identification
* 💡 Gemini AI Career Recommendations
* 🔐 Role-Based Access Control (ACLs)
* 📈 Dashboard & Analytics

## 🏗️ Built With

* ServiceNow App Engine Studio
* JavaScript
* Business Rules
* ACLs
* Reports & Dashboards
* Gemini AI API
